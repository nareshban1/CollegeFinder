106def splprep(x, w=None, u=None, ub=None, ue=None, k=3, task=0, s=None, t=None, 

107 full_output=0, nest=None, per=0, quiet=1): 

108 """ 

109 Find the B-spline representation of an N-dimensional curve. 

110 

111 Given a list of N rank-1 arrays, `x`, which represent a curve in 

112 N-dimensional space parametrized by `u`, find a smooth approximating 

113 spline curve g(`u`). Uses the FORTRAN routine parcur from FITPACK. 

114 

115 Parameters 

116 ---------- 

117 x : array_like 

118 A list of sample vector arrays representing the curve. 

119 w : array_like, optional 

120 Strictly positive rank-1 array of weights the same length as `x[0]`. 

121 The weights are used in computing the weighted least-squares spline 

122 fit. If the errors in the `x` values have standard-deviation given by 

123 the vector d, then `w` should be 1/d. Default is ``ones(len(x[0]))``. 

124 u : array_like, optional 

125 An array of parameter values. If not given, these values are 

126 calculated automatically as ``M = len(x[0])``, where 

127 

128 v[0] = 0 

129 

130 v[i] = v[i-1] + distance(`x[i]`, `x[i-1]`) 

131 

132 u[i] = v[i] / v[M-1] 

133 

134 ub, ue : int, optional 

135 The end-points of the parameters interval. Defaults to 

136 u[0] and u[-1]. 

137 k : int, optional 

138 Degree of the spline. Cubic splines are recommended. 

139 Even values of `k` should be avoided especially with a small s-value. 

140 ``1 <= k <= 5``, default is 3. 

141 task : int, optional 

142 If task==0 (default), find t and c for a given smoothing factor, s. 

143 If task==1, find t and c for another value of the smoothing factor, s. 

144 There must have been a previous call with task=0 or task=1 

145 for the same set of data. 

146 If task=-1 find the weighted least square spline for a given set of 

147 knots, t. 

148 s : float, optional 

149 A smoothing condition. The amount of smoothness is determined by 

150 satisfying the conditions: ``sum((w * (y - g))**2,axis=0) <= s``, 

151 where g(x) is the smoothed interpolation of (x,y). The user can 

152 use `s` to control the trade-off between closeness and smoothness 

153 of fit. Larger `s` means more smoothing while smaller values of `s` 

154 indicate less smoothing. Recommended values of `s` depend on the 

155 weights, w. If the weights represent the inverse of the 

156 standard-deviation of y, then a good `s` value should be found in 

157 the range ``(m-sqrt(2*m),m+sqrt(2*m))``, where m is the number of 

158 data points in x, y, and w. 

159 t : int, optional 

160 The knots needed for task=-1. 

161 full_output : int, optional 

162 If non-zero, then return optional outputs. 

163 nest : int, optional 

164 An over-estimate of the total number of knots of the spline to 

165 help in determining the storage space. By default nest=m/2. 

166 Always large enough is nest=m+k+1. 

167 per : int, optional 

168 If non-zero, data points are considered periodic with period 

169 ``x[m-1] - x[0]`` and a smooth periodic spline approximation is 

170 returned. Values of ``y[m-1]`` and ``w[m-1]`` are not used. 

171 quiet : int, optional 

172 Non-zero to suppress messages. 

173 This parameter is deprecated; use standard Python warning filters 

174 instead. 

175 

176 Returns 

177 ------- 

178 tck : tuple 

179 A tuple (t,c,k) containing the vector of knots, the B-spline 

180 coefficients, and the degree of the spline. 

181 u : array 

182 An array of the values of the parameter. 

183 fp : float 

184 The weighted sum of squared residuals of the spline approximation. 

185 ier : int 

186 An integer flag about splrep success. Success is indicated 

187 if ier<=0. If ier in [1,2,3] an error occurred but was not raised. 

188 Otherwise an error is raised. 

189 msg : str 

190 A message corresponding to the integer flag, ier. 

191 

192 See Also 

193 -------- 

194 splrep, splev, sproot, spalde, splint, 

195 bisplrep, bisplev 

196 UnivariateSpline, BivariateSpline 

197 

198 Notes 

199 ----- 

200 See `splev` for evaluation of the spline and its derivatives. 

201 The number of dimensions N must be smaller than 11. 

202 

203 References 

204 ---------- 

205 .. [1] P. Dierckx, "Algorithms for smoothing data with periodic and 

206 parametric splines, Computer Graphics and Image Processing", 

207 20 (1982) 171-184. 

208 .. [2] P. Dierckx, "Algorithms for smoothing data with periodic and 

209 parametric splines", report tw55, Dept. Computer Science, 

210 K.U.Leuven, 1981. 

211 .. [3] P. Dierckx, "Curve and surface fitting with splines", Monographs on 

212 Numerical Analysis, Oxford University Press, 1993. 

213 

214 """ 

215 if task <= 0: 

216 _parcur_cache = {'t': array([], float), 'wrk': array([], float), 

217 'iwrk': array([], intc), 'u': array([], float), 

218 'ub': 0, 'ue': 1} 

219 x = atleast_1d(x) 

220 idim, m = x.shape 

221 if per: 

222 for i in range(idim): 

223 if x[i][0] != x[i][-1]: 

224 if quiet < 2: 

225 warnings.warn(RuntimeWarning('Setting x[%d][%d]=x[%d][0]' % 

226 (i, m, i))) 

227 x[i][-1] = x[i][0] 

228 if not 0 < idim < 11: 

229 raise TypeError('0 < idim < 11 must hold') 

230 if w is None: 

231 w = ones(m, float) 

232 else: 

233 w = atleast_1d(w) 

234 ipar = (u is not None) 

235 if ipar: 

236 _parcur_cache['u'] = u 

237 if ub is None: 

238 _parcur_cache['ub'] = u[0] 

239 else: 

240 _parcur_cache['ub'] = ub 

241 if ue is None: 

242 _parcur_cache['ue'] = u[-1] 

243 else: 

244 _parcur_cache['ue'] = ue 

245 else: 

246 _parcur_cache['u'] = zeros(m, float) 

247 if not (1 <= k <= 5): 

248 raise TypeError('1 <= k= %d <=5 must hold' % k) 

249 if not (-1 <= task <= 1): 

250 raise TypeError('task must be -1, 0 or 1') 

251 if (not len(w) == m) or (ipar == 1 and (not len(u) == m)): 

252 raise TypeError('Mismatch of input dimensions') 

253 if s is None: 

254 s = m - sqrt(2*m) 

255 if t is None and task == -1: 

256 raise TypeError('Knots must be given for task=-1') 

257 if t is not None: 

258 _parcur_cache['t'] = atleast_1d(t) 

259 n = len(_parcur_cache['t']) 

260 if task == -1 and n < 2*k + 2: 

261 raise TypeError('There must be at least 2*k+2 knots for task=-1') 

262 if m <= k: 

263 raise TypeError('m > k must hold') 

264 if nest is None: 

265 nest = m + 2*k 

266 

267 if (task >= 0 and s == 0) or (nest < 0): 

268 if per: 

269 nest = m + 2*k 

270 else: 

271 nest = m + k + 1 

272 nest = max(nest, 2*k + 3) 

273 u = _parcur_cache['u'] 

274 ub = _parcur_cache['ub'] 

275 ue = _parcur_cache['ue'] 

276 t = _parcur_cache['t'] 

277 wrk = _parcur_cache['wrk'] 

278 iwrk = _parcur_cache['iwrk'] 

279 t, c, o = _fitpack._parcur(ravel(transpose(x)), w, u, ub, ue, k, 

280 task, ipar, s, t, nest, wrk, iwrk, per) 

281 _parcur_cache['u'] = o['u'] 

282 _parcur_cache['ub'] = o['ub'] 

283 _parcur_cache['ue'] = o['ue'] 

284 _parcur_cache['t'] = t 

285 _parcur_cache['wrk'] = o['wrk'] 

286 _parcur_cache['iwrk'] = o['iwrk'] 

287 ier = o['ier'] 

288 fp = o['fp'] 

289 n = len(t) 

290 u = o['u'] 

291 c.shape = idim, n - k - 1 

292 tcku = [t, list(c), k], u 

293 if ier <= 0 and not quiet: 

294 warnings.warn(RuntimeWarning(_iermess[ier][0] + 

295 "\tk=%d n=%d m=%d fp=%f s=%f" % 

296 (k, len(t), m, fp, s))) 

297 if ier > 0 and not full_output: 

298 if ier in [1, 2, 3]: 

299 warnings.warn(RuntimeWarning(_iermess[ier][0])) 

300 else: 

301 try: 

302 raise _iermess[ier][1](_iermess[ier][0]) 

303 except KeyError: 

304 raise _iermess['unknown'][1](_iermess['unknown'][0]) 

305 if full_output: 

306 try: 

307 return tcku, fp, ier, _iermess[ier][0] 

308 except KeyError: 

309 return tcku, fp, ier, _iermess['unknown'][0] 

310 else: 

311 return tcku

318def splrep(x, y, w=None, xb=None, xe=None, k=3, task=0, s=None, t=None, 

319 full_output=0, per=0, quiet=1): 

320 """ 

321 Find the B-spline representation of 1-D curve. 

322 

323 Given the set of data points ``(x[i], y[i])`` determine a smooth spline 

324 approximation of degree k on the interval ``xb <= x <= xe``. 

325 

326 Parameters 

327 ---------- 

328 x, y : array_like 

329 The data points defining a curve y = f(x). 

330 w : array_like, optional 

331 Strictly positive rank-1 array of weights the same length as x and y. 

332 The weights are used in computing the weighted least-squares spline 

333 fit. If the errors in the y values have standard-deviation given by the 

334 vector d, then w should be 1/d. Default is ones(len(x)). 

335 xb, xe : float, optional 

336 The interval to fit. If None, these default to x[0] and x[-1] 

337 respectively. 

338 k : int, optional 

339 The order of the spline fit. It is recommended to use cubic splines. 

340 Even order splines should be avoided especially with small s values. 

341 1 <= k <= 5 

342 task : {1, 0, -1}, optional 

343 If task==0 find t and c for a given smoothing factor, s. 

344 

345 If task==1 find t and c for another value of the smoothing factor, s. 

346 There must have been a previous call with task=0 or task=1 for the same 

347 set of data (t will be stored an used internally) 

348 

349 If task=-1 find the weighted least square spline for a given set of 

350 knots, t. These should be interior knots as knots on the ends will be 

351 added automatically. 

352 s : float, optional 

353 A smoothing condition. The amount of smoothness is determined by 

354 satisfying the conditions: sum((w * (y - g))**2,axis=0) <= s where g(x) 

355 is the smoothed interpolation of (x,y). The user can use s to control 

356 the tradeoff between closeness and smoothness of fit. Larger s means 

357 more smoothing while smaller values of s indicate less smoothing. 

358 Recommended values of s depend on the weights, w. If the weights 

359 represent the inverse of the standard-deviation of y, then a good s 

360 value should be found in the range (m-sqrt(2*m),m+sqrt(2*m)) where m is 

361 the number of datapoints in x, y, and w. default : s=m-sqrt(2*m) if 

362 weights are supplied. s = 0.0 (interpolating) if no weights are 

363 supplied. 

364 t : array_like, optional 

365 The knots needed for task=-1. If given then task is automatically set 

366 to -1. 

367 full_output : bool, optional 

368 If non-zero, then return optional outputs. 

369 per : bool, optional 

370 If non-zero, data points are considered periodic with period x[m-1] - 

371 x[0] and a smooth periodic spline approximation is returned. Values of 

372 y[m-1] and w[m-1] are not used. 

373 quiet : bool, optional 

374 Non-zero to suppress messages. 

375 This parameter is deprecated; use standard Python warning filters 

376 instead. 

377 

378 Returns 

379 ------- 

380 tck : tuple 

381 (t,c,k) a tuple containing the vector of knots, the B-spline 

382 coefficients, and the degree of the spline. 

383 fp : array, optional 

384 The weighted sum of squared residuals of the spline approximation. 

385 ier : int, optional 

386 An integer flag about splrep success. Success is indicated if ier<=0. 

387 If ier in [1,2,3] an error occurred but was not raised. Otherwise an 

388 error is raised. 

389 msg : str, optional 

390 A message corresponding to the integer flag, ier. 

391 

392 Notes 

393 ----- 

394 See splev for evaluation of the spline and its derivatives. 

395 

396 The user is responsible for assuring that the values of *x* are unique. 

397 Otherwise, *splrep* will not return sensible results. 

398 

399 See Also 

400 -------- 

401 UnivariateSpline, BivariateSpline 

402 splprep, splev, sproot, spalde, splint 

403 bisplrep, bisplev 

404 

405 Notes 

406 ----- 

407 See splev for evaluation of the spline and its derivatives. Uses the 

408 FORTRAN routine curfit from FITPACK. 

409 

410 If provided, knots `t` must satisfy the Schoenberg-Whitney conditions, 

411 i.e., there must be a subset of data points ``x[j]`` such that 

412 ``t[j] < x[j] < t[j+k+1]``, for ``j=0, 1,...,n-k-2``. 

413 

414 References 

415 ---------- 

416 Based on algorithms described in [1]_, [2]_, [3]_, and [4]_: 

417 

418 .. [1] P. Dierckx, "An algorithm for smoothing, differentiation and 

419 integration of experimental data using spline functions", 

420 J.Comp.Appl.Maths 1 (1975) 165-184. 

421 .. [2] P. Dierckx, "A fast algorithm for smoothing data on a rectangular 

422 grid while using spline functions", SIAM J.Numer.Anal. 19 (1982) 

423 1286-1304. 

424 .. [3] P. Dierckx, "An improved algorithm for curve fitting with spline 

425 functions", report tw54, Dept. Computer Science,K.U. Leuven, 1981. 

426 .. [4] P. Dierckx, "Curve and surface fitting with splines", Monographs on 

427 Numerical Analysis, Oxford University Press, 1993. 

428 

429 Examples 

430 -------- 

431 

432 >>> import matplotlib.pyplot as plt 

433 >>> from scipy.interpolate import splev, splrep 

434 >>> x = np.linspace(0, 10, 10) 

435 >>> y = np.sin(x) 

436 >>> tck = splrep(x, y) 

437 >>> x2 = np.linspace(0, 10, 200) 

438 >>> y2 = splev(x2, tck) 

439 >>> plt.plot(x, y, 'o', x2, y2) 

440 >>> plt.show() 

441 

442 """ 

443 if task <= 0: 

444 _curfit_cache = {} 

445 x, y = map(atleast_1d, [x, y]) 

446 m = len(x) 

447 if w is None: 

448 w = ones(m, float) 

449 if s is None: 

450 s = 0.0 

451 else: 

452 w = atleast_1d(w) 

453 if s is None: 

454 s = m - sqrt(2*m) 

455 if not len(w) == m: 

456 raise TypeError('len(w)=%d is not equal to m=%d' % (len(w), m)) 

457 if (m != len(y)) or (m != len(w)): 

458 raise TypeError('Lengths of the first three arguments (x,y,w) must ' 

459 'be equal') 

460 if not (1 <= k <= 5): 

461 raise TypeError('Given degree of the spline (k=%d) is not supported. ' 

462 '(1<=k<=5)' % k) 

463 if m <= k: 

464 raise TypeError('m > k must hold') 

465 if xb is None: 

466 xb = x[0] 

467 if xe is None: 

468 xe = x[-1] 

469 if not (-1 <= task <= 1): 

470 raise TypeError('task must be -1, 0 or 1') 

471 if t is not None: 

472 task = -1 

473 if task == -1: 

474 if t is None: 

475 raise TypeError('Knots must be given for task=-1') 

476 numknots = len(t) 

477 _curfit_cache['t'] = empty((numknots + 2*k + 2,), float) 

478 _curfit_cache['t'][k+1:-k-1] = t 

479 nest = len(_curfit_cache['t']) 

480 elif task == 0: 

481 if per: 

482 nest = max(m + 2*k, 2*k + 3) 

483 else: 

484 nest = max(m + k + 1, 2*k + 3) 

485 t = empty((nest,), float) 

486 _curfit_cache['t'] = t 

487 if task <= 0: 

488 if per: 

489 _curfit_cache['wrk'] = empty((m*(k + 1) + nest*(8 + 5*k),), float) 

490 else: 

491 _curfit_cache['wrk'] = empty((m*(k + 1) + nest*(7 + 3*k),), float) 

492 _curfit_cache['iwrk'] = empty((nest,), intc) 

493 try: 

494 t = _curfit_cache['t'] 

495 wrk = _curfit_cache['wrk'] 

496 iwrk = _curfit_cache['iwrk'] 

497 except KeyError: 

498 raise TypeError("must call with task=1 only after" 

499 " call with task=0,-1") 

500 if not per: 

501 n, c, fp, ier = dfitpack.curfit(task, x, y, w, t, wrk, iwrk, 

502 xb, xe, k, s) 

503 else: 

504 n, c, fp, ier = dfitpack.percur(task, x, y, w, t, wrk, iwrk, k, s) 

505 tck = (t[:n], c[:n], k) 

506 if ier <= 0 and not quiet: 

507 _mess = (_iermess[ier][0] + "\tk=%d n=%d m=%d fp=%f s=%f" % 

508 (k, len(t), m, fp, s)) 

509 warnings.warn(RuntimeWarning(_mess)) 

510 if ier > 0 and not full_output: 

511 if ier in [1, 2, 3]: 

512 warnings.warn(RuntimeWarning(_iermess[ier][0])) 

513 else: 

514 try: 

515 raise _iermess[ier][1](_iermess[ier][0]) 

516 except KeyError: 

517 raise _iermess['unknown'][1](_iermess['unknown'][0]) 

518 if full_output: 

519 try: 

520 return tck, fp, ier, _iermess[ier][0] 

521 except KeyError: 

522 return tck, fp, ier, _iermess['unknown'][0] 

523 else: 

524 return tck

527def splev(x, tck, der=0, ext=0): 

528 """ 

529 Evaluate a B-spline or its derivatives. 

530 

531 Given the knots and coefficients of a B-spline representation, evaluate 

532 the value of the smoothing polynomial and its derivatives. This is a 

533 wrapper around the FORTRAN routines splev and splder of FITPACK. 

534 

535 Parameters 

536 ---------- 

537 x : array_like 

538 An array of points at which to return the value of the smoothed 

539 spline or its derivatives. If `tck` was returned from `splprep`, 

540 then the parameter values, u should be given. 

541 tck : tuple 

542 A sequence of length 3 returned by `splrep` or `splprep` containing 

543 the knots, coefficients, and degree of the spline. 

544 der : int, optional 

545 The order of derivative of the spline to compute (must be less than 

546 or equal to k). 

547 ext : int, optional 

548 Controls the value returned for elements of ``x`` not in the 

549 interval defined by the knot sequence. 

550 

551 * if ext=0, return the extrapolated value. 

552 * if ext=1, return 0 

553 * if ext=2, raise a ValueError 

554 * if ext=3, return the boundary value. 

555 

556 The default value is 0. 

557 

558 Returns 

559 ------- 

560 y : ndarray or list of ndarrays

801def bisplrep(x, y, z, w=None, xb=None, xe=None, yb=None, ye=None, 

802 kx=3, ky=3, task=0, s=None, eps=1e-16, tx=None, ty=None, 

803 full_output=0, nxest=None, nyest=None, quiet=1): 

804 """ 

805 Find a bivariate B-spline representation of a surface. 

806 

807 Given a set of data points (x[i], y[i], z[i]) representing a surface 

808 z=f(x,y), compute a B-spline representation of the surface. Based on 

809 the routine SURFIT from FITPACK. 

810 

811 Parameters 

812 ---------- 

813 x, y, z : ndarray 

814 Rank-1 arrays of data points. 

815 w : ndarray, optional 

816 Rank-1 array of weights. By default ``w=np.ones(len(x))``. 

817 xb, xe : float, optional 

818 End points of approximation interval in `x`. 

819 By default ``xb = x.min(), xe=x.max()``. 

820 yb, ye : float, optional 

821 End points of approximation interval in `y`. 

822 By default ``yb=y.min(), ye = y.max()``. 

823 kx, ky : int, optional 

824 The degrees of the spline (1 <= kx, ky <= 5). 

825 Third order (kx=ky=3) is recommended. 

826 task : int, optional 

827 If task=0, find knots in x and y and coefficients for a given 

828 smoothing factor, s. 

829 If task=1, find knots and coefficients for another value of the 

830 smoothing factor, s. bisplrep must have been previously called 

831 with task=0 or task=1. 

832 If task=-1, find coefficients for a given set of knots tx, ty. 

833 s : float, optional 

834 A non-negative smoothing factor. If weights correspond 

835 to the inverse of the standard-deviation of the errors in z, 

836 then a good s-value should be found in the range 

837 ``(m-sqrt(2*m),m+sqrt(2*m))`` where m=len(x). 

838 eps : float, optional 

839 A threshold for determining the effective rank of an 

840 over-determined linear system of equations (0 < eps < 1). 

841 `eps` is not likely to need changing. 

842 tx, ty : ndarray, optional 

843 Rank-1 arrays of the knots of the spline for task=-1 

844 full_output : int, optional 

845 Non-zero to return optional outputs. 

846 nxest, nyest : int, optional 

847 Over-estimates of the total number of knots. If None then 

848 ``nxest = max(kx+sqrt(m/2),2*kx+3)``, 

849 ``nyest = max(ky+sqrt(m/2),2*ky+3)``. 

850 quiet : int, optional 

851 Non-zero to suppress printing of messages. 

852 This parameter is deprecated; use standard Python warning filters 

853 instead. 

854 

855 Returns 

856 ------- 

857 tck : array_like 

858 A list [tx, ty, c, kx, ky] containing the knots (tx, ty) and 

859 coefficients (c) of the bivariate B-spline representation of the 

860 surface along with the degree of the spline. 

861 fp : ndarray 

862 The weighted sum of squared residuals of the spline approximation. 

863 ier : int 

864 An integer flag about splrep success. Success is indicated if 

865 ier<=0. If ier in [1,2,3] an error occurred but was not raised. 

866 Otherwise an error is raised. 

867 msg : str 

868 A message corresponding to the integer flag, ier. 

869 

870 See Also 

871 -------- 

872 splprep, splrep, splint, sproot, splev 

873 UnivariateSpline, BivariateSpline 

874 

875 Notes 

876 ----- 

877 See `bisplev` to evaluate the value of the B-spline given its tck 

878 representation. 

879 

880 References 

881 ---------- 

882 .. [1] Dierckx P.:An algorithm for surface fitting with spline functions 

883 Ima J. Numer. Anal. 1 (1981) 267-283. 

884 .. [2] Dierckx P.:An algorithm for surface fitting with spline functions 

885 report tw50, Dept. Computer Science,K.U.Leuven, 1980. 

886 .. [3] Dierckx P.:Curve and surface fitting with splines, Monographs on 

887 Numerical Analysis, Oxford University Press, 1993. 

888 

889 """ 

890 x, y, z = map(ravel, [x, y, z]) # ensure 1-d arrays. 

891 m = len(x) 

892 if not (m == len(y) == len(z)): 

893 raise TypeError('len(x)==len(y)==len(z) must hold.') 

894 if w is None: 

895 w = ones(m, float) 

896 else: 

897 w = atleast_1d(w) 

898 if not len(w) == m: 

899 raise TypeError('len(w)=%d is not equal to m=%d' % (len(w), m)) 

900 if xb is None: 

901 xb = x.min() 

902 if xe is None: 

903 xe = x.max() 

904 if yb is None: 

905 yb = y.min() 

906 if ye is None: 

907 ye = y.max() 

908 if not (-1 <= task <= 1): 

909 raise TypeError('task must be -1, 0 or 1') 

910 if s is None: 

911 s = m - sqrt(2*m) 

912 if tx is None and task == -1: 

913 raise TypeError('Knots_x must be given for task=-1') 

914 if tx is not None: 

915 _surfit_cache['tx'] = atleast_1d(tx) 

916 nx = len(_surfit_cache['tx']) 

917 if ty is None and task == -1: 

918 raise TypeError('Knots_y must be given for task=-1') 

919 if ty is not None: 

920 _surfit_cache['ty'] = atleast_1d(ty) 

921 ny = len(_surfit_cache['ty']) 

922 if task == -1 and nx < 2*kx+2: 

923 raise TypeError('There must be at least 2*kx+2 knots_x for task=-1') 

924 if task == -1 and ny < 2*ky+2: 

925 raise TypeError('There must be at least 2*ky+2 knots_x for task=-1') 

926 if not ((1 <= kx <= 5) and (1 <= ky <= 5)): 

927 raise TypeError('Given degree of the spline (kx,ky=%d,%d) is not ' 

928 'supported. (1<=k<=5)' % (kx, ky)) 

929 if m < (kx + 1)*(ky + 1): 

930 raise TypeError('m >= (kx+1)(ky+1) must hold') 

931 if nxest is None: 

932 nxest = int(kx + sqrt(m/2)) 

933 if nyest is None: 

934 nyest = int(ky + sqrt(m/2)) 

935 nxest, nyest = max(nxest, 2*kx + 3), max(nyest, 2*ky + 3) 

936 if task >= 0 and s == 0: 

937 nxest = int(kx + sqrt(3*m)) 

938 nyest = int(ky + sqrt(3*m)) 

939 if task == -1: 

940 _surfit_cache['tx'] = atleast_1d(tx) 

941 _surfit_cache['ty'] = atleast_1d(ty) 

942 tx, ty = _surfit_cache['tx'], _surfit_cache['ty'] 

943 wrk = _surfit_cache['wrk'] 

944 u = nxest - kx - 1 

945 v = nyest - ky - 1 

946 km = max(kx, ky) + 1 

947 ne = max(nxest, nyest) 

948 bx, by = kx*v + ky + 1, ky*u + kx + 1 

949 b1, b2 = bx, bx + v - ky 

950 if bx > by: 

951 b1, b2 = by, by + u - kx 

952 msg = "Too many data points to interpolate" 

953 lwrk1 = _intc_overflow(u*v*(2 + b1 + b2) + 

954 2*(u + v + km*(m + ne) + ne - kx - ky) + b2 + 1, 

955 msg=msg) 

956 lwrk2 = _intc_overflow(u*v*(b2 + 1) + b2, msg=msg) 

957 tx, ty, c, o = _fitpack._surfit(x, y, z, w, xb, xe, yb, ye, kx, ky, 

958 task, s, eps, tx, ty, nxest, nyest, 

959 wrk, lwrk1, lwrk2) 

960 _curfit_cache['tx'] = tx 

961 _curfit_cache['ty'] = ty 

962 _curfit_cache['wrk'] = o['wrk'] 

963 ier, fp = o['ier'], o['fp'] 

964 tck = [tx, ty, c, kx, ky] 

965 

966 ierm = min(11, max(-3, ier)) 

967 if ierm <= 0 and not quiet: 

968 _mess = (_iermess2[ierm][0] + 

969 "\tkx,ky=%d,%d nx,ny=%d,%d m=%d fp=%f s=%f" % 

970 (kx, ky, len(tx), len(ty), m, fp, s)) 

971 warnings.warn(RuntimeWarning(_mess)) 

972 if ierm > 0 and not full_output: 

973 if ier in [1, 2, 3, 4, 5]: 

974 _mess = ("\n\tkx,ky=%d,%d nx,ny=%d,%d m=%d fp=%f s=%f" % 

975 (kx, ky, len(tx), len(ty), m, fp, s)) 

976 warnings.warn(RuntimeWarning(_iermess2[ierm][0] + _mess)) 

977 else: 

978 try: 

979 raise _iermess2[ierm][1](_iermess2[ierm][0]) 

980 except KeyError: 

981 raise _iermess2['unknown'][1](_iermess2['unknown'][0]) 

982 if full_output: 

983 try: 

984 return tck, fp, ier, _iermess2[ierm][0] 

985 except KeyError: 

986 return tck, fp, ier, _iermess2['unknown'][0] 

987 else: 

988 return tck

991def bisplev(x, y, tck, dx=0, dy=0): 

992 """ 

993 Evaluate a bivariate B-spline and its derivatives. 

994 

995 Return a rank-2 array of spline function values (or spline derivative 

996 values) at points given by the cross-product of the rank-1 arrays `x` and 

997 `y`. In special cases, return an array or just a float if either `x` or 

998 `y` or both are floats. Based on BISPEV from FITPACK. 

999 

1000 Parameters 

1001 ---------- 

1002 x, y : ndarray 

1003 Rank-1 arrays specifying the domain over which to evaluate the 

1004 spline or its derivative. 

1005 tck : tuple 

1006 A sequence of length 5 returned by `bisplrep` containing the knot 

1007 locations, the coefficients, and the degree of the spline: 

1008 [tx, ty, c, kx, ky]. 

1009 dx, dy : int, optional 

1010 The orders of the partial derivatives in `x` and `y` respectively. 

1011 

1012 Returns 

1013 ------- 

1014 vals : ndarray 

1015 The B-spline or its derivative evaluated over the set formed by 

1016 the cross-product of `x` and `y`. 

1017 

1018 See Also 

1019 -------- 

1020 splprep, splrep, splint, sproot, splev 

1021 UnivariateSpline, BivariateSpline 

1022 

1023 Notes 

1024 ----- 

1025 See `bisplrep` to generate the `tck` representation. 

1026 

1027 References 

1028 ---------- 

1029 .. [1] Dierckx P. : An algorithm for surface fitting 

1030 with spline functions

1032 .. [2] Dierckx P. : An algorithm for surface fitting 

1033 with spline functions 

1034 report tw50, Dept. Computer Science,K.U.Leuven, 1980. 

1035 .. [3] Dierckx P. : Curve and surface fitting with splines, 

1036 Monographs on Numerical Analysis, Oxford University Press, 1993. 

1037 

1038 """ 

1039 tx, ty, c, kx, ky = tck 

1040 if not (0 <= dx < kx): 

1041 raise ValueError("0 <= dx = %d < kx = %d must hold" % (dx, kx)) 

1042 if not (0 <= dy < ky): 

1043 raise ValueError("0 <= dy = %d < ky = %d must hold" % (dy, ky)) 

1044 x, y = map(atleast_1d, [x, y]) 

1045 if (len(x.shape) != 1) or (len(y.shape) != 1): 

1046 raise ValueError("First two entries should be rank-1 arrays.")

1049 raise ValueError("Invalid input data") 

1050 if ier: 

1051 raise TypeError("An error occurred")

1053 if len(z) > 1: 

1054 return z 

1055 if len(z[0]) > 1: 

1056 return z[0] 

1057 return z[0][0]

1083def insert(x, tck, m=1, per=0): 

1084 """ 

1085 Insert knots into a B-spline. 

1086 

1087 Given the knots and coefficients of a B-spline representation, create a 

1088 new B-spline with a knot inserted `m` times at point `x`. 

1089 This is a wrapper around the FORTRAN routine insert of FITPACK. 

1090 

1091 Parameters 

1092 ---------- 

1093 x (u) : array_like 

1094 A 1-D point at which to insert a new knot(s). If `tck` was returned 

1095 from ``splprep``, then the parameter values, u should be given. 

1096 tck : tuple 

1097 A tuple (t,c,k) returned by ``splrep`` or ``splprep`` containing 

1098 the vector of knots, the B-spline coefficients, 

1099 and the degree of the spline. 

1100 m : int, optional 

1101 The number of times to insert the given knot (its multiplicity). 

1102 Default is 1. 

1103 per : int, optional 

1104 If non-zero, the input spline is considered periodic. 

1105 

1106 Returns 

1107 ------- 

1108 tck : tuple 

1109 A tuple (t,c,k) containing the vector of knots, the B-spline 

1110 coefficients, and the degree of the new spline. 

1111 ``t(k+1) <= x <= t(n-k)``, where k is the degree of the spline. 

1112 In case of a periodic spline (``per != 0``) there must be 

1113 either at least k interior knots t(j) satisfying ``t(k+1)<t(j)<=x`` 

1114 or at least k interior knots t(j) satisfying ``x<=t(j)<t(n-k)``. 

1115 

1116 Notes 

1117 ----- 

1118 Based on algorithms from [1]_ and [2]_. 

1119 

1120 References 

1121 ---------- 

1122 .. [1] W. Boehm, "Inserting new knots into b-spline curves.", 

1123 Computer Aided Design, 12, p.199-201, 1980. 

1124 .. [2] P. Dierckx, "Curve and surface fitting with splines, Monographs on 

1125 Numerical Analysis", Oxford University Press, 1993. 

1126 

1127 """ 

1128 t, c, k = tck 

1129 try: 

1130 c[0][0] 

1131 parametric = True 

1132 except Exception: 

1133 parametric = False 

1134 if parametric: 

1135 cc = [] 

1136 for c_vals in c: 

1137 tt, cc_val, kk = insert(x, [t, c_vals, k], m) 

1138 cc.append(cc_val) 

1139 return (tt, cc, kk) 

1140 else: 

1141 tt, cc, ier = _fitpack._insert(per, t, c, k, x, m) 

1142 if ier == 10: 

1143 raise ValueError("Invalid input data") 

1144 if ier: 

1145 raise TypeError("An error occurred") 

1146 return (tt, cc, k)

1150 """ 

1151 Compute the spline representation of the derivative of a given spline 

1152 

1153 Parameters

1156 Spline whose derivative to compute

1175 Examples 

1176 -------- 

1177 This can be used for finding maxima of a curve: 

1178 

1179 >>> from scipy.interpolate import splrep, splder, sproot 

1180 >>> x = np.linspace(0, 10, 70) 

1181 >>> y = np.sin(x) 

1182 >>> spl = splrep(x, y, k=4) 

1183 

1184 Now, differentiate the spline and find the zeros of the 

1185 derivative. (NB: `sproot` only works for order 3 splines, so we 

1186 fit an order 4 spline):

1188 >>> dspl = splder(spl)

1191 

1192 This agrees well with roots :math:`\\pi/2 + n\\pi` of 

1193 :math:`\\cos(x) = \\sin'(x)`. 

1194 

1195 """ 

1196 if n < 0: 

1197 return splantider(tck, -n) 

1198 

1199 t, c, k = tck 

1200 

1201 if n > k: 

1202 raise ValueError(("Order of derivative (n = %r) must be <= " 

1203 "order of spline (k = %r)") % (n, tck[2])) 

1204 

1205 # Extra axes for the trailing dims of the `c` array: 

1206 sh = (slice(None),) + ((None,)*len(c.shape[1:])) 

1207 

1208 with np.errstate(invalid='raise', divide='raise'): 

1209 try: 

1210 for j in range(n): 

1211 # See e.g. Schumaker, Spline Functions: Basic Theory, Chapter 5 

1212 

1213 # Compute the denominator in the differentiation formula. 

1214 # (and append traling dims, if necessary) 

1215 dt = t[k+1:-1] - t[1:-k-1] 

1216 dt = dt[sh] 

1217 # Compute the new coefficients 

1218 c = (c[1:-1-k] - c[:-2-k]) * k / dt 

1219 # Pad coefficient array to same size as knots (FITPACK 

1220 # convention) 

1221 c = np.r_[c, np.zeros((k,) + c.shape[1:])] 

1222 # Adjust knots 

1223 t = t[1:-1] 

1224 k -= 1 

1225 except FloatingPointError: 

1226 raise ValueError(("The spline has internal repeated knots " 

1227 "and is not differentiable %d times") % n) 

1228 

1229 return t, c, k

1232def splantider(tck, n=1): 

1233 """ 

1234 Compute the spline for the antiderivative (integral) of a given spline. 

1235 

1236 Parameters 

1237 ---------- 

1238 tck : tuple of (t, c, k) 

1239 Spline whose antiderivative to compute 

1240 n : int, optional 

1241 Order of antiderivative to evaluate. Default: 1 

1242 

1243 Returns 

1244 ------- 

1245 tck_ader : tuple of (t2, c2, k2) 

1246 Spline of order k2=k+n representing the antiderivative of the input 

1247 spline. 

1248 

1249 See Also 

1250 -------- 

1251 splder, splev, spalde 

1252 

1253 Notes 

1254 ----- 

1255 The `splder` function is the inverse operation of this function. 

1256 Namely, ``splder(splantider(tck))`` is identical to `tck`, modulo 

1257 rounding error. 

1258 

1259 .. versionadded:: 0.13.0 

1260 

1261 Examples 

1262 -------- 

1263 >>> from scipy.interpolate import splrep, splder, splantider, splev 

1264 >>> x = np.linspace(0, np.pi/2, 70) 

1265 >>> y = 1 / np.sqrt(1 - 0.8*np.sin(x)**2) 

1266 >>> spl = splrep(x, y) 

1267 

1268 The derivative is the inverse operation of the antiderivative, 

1269 although some floating point error accumulates: 

1270 

1271 >>> splev(1.7, spl), splev(1.7, splder(splantider(spl))) 

1272 (array(2.1565429877197317), array(2.1565429877201865)) 

1273 

1274 Antiderivative can be used to evaluate definite integrals: 

1275 

1276 >>> ispl = splantider(spl) 

1277 >>> splev(np.pi/2, ispl) - splev(0, ispl) 

1278 2.2572053588768486 

1279 

1280 This is indeed an approximation to the complete elliptic integral 

1281 :math:`K(m) = \\int_0^{\\pi/2} [1 - m\\sin^2 x]^{-1/2} dx`: 

1282 

1283 >>> from scipy.special import ellipk 

1284 >>> ellipk(0.8) 

1285 2.2572053268208538 

1286 

1287 """ 

1288 if n < 0: 

1289 return splder(tck, -n) 

1290 

1291 t, c, k = tck 

1292 

1293 # Extra axes for the trailing dims of the `c` array: 

1294 sh = (slice(None),) + (None,)*len(c.shape[1:]) 

1295 

1296 for j in range(n): 

1297 # This is the inverse set of operations to splder. 

1298 

1299 # Compute the multiplier in the antiderivative formula. 

1300 dt = t[k+1:] - t[:-k-1] 

1301 dt = dt[sh] 

1302 # Compute the new coefficients 

1303 c = np.cumsum(c[:-k-1] * dt, axis=0) / (k + 1) 

1304 c = np.r_[np.zeros((1,) + c.shape[1:]), 

1305 c, 

1306 [c[-1]] * (k+2)] 

1307 # New knots 

1308 t = np.r_[t[0], t, t[-1]] 

1309 k += 1 

1310 

1311 return t, c, k
